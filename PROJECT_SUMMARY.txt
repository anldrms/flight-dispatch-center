🚀 QUICK START
   1. Start server:    npm start
   2. Open browser:    http://localhost:3000
   3. Plan flight:     Enter departure/arrival
   4. Calculate:       Click "Calculate Route"
   5. Export:          Generate PDF or copy route

🆘 TROUBLESHOOTING
   Server won't start:
   → npm install && npm start
   
   Can't connect:
   → Check server is running
   → Try http://127.0.0.1:3000
   → Clear browser cache
   
   No weather data:
   → Check internet connection
   → Try major airports (KJFK, EGLL)
   → Some airports don't have METAR
